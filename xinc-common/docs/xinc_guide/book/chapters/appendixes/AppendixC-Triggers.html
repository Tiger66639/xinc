Appendix C: Triggers

This appendix contains a reference of all trigger tasks, i.e. all tasks that are needed to start a build.

<schedule/>

The Schedule Task sets the interval in which repository changes are looked up by the <modificationset/>.

<project name="ProjectName">
    <trigger>
        <schedule interval="300"/>
    </trigger>
...
</project>

Parent Tags

  • <trigger/>

Parameters

Name Type Description Default Required
interval Integer Number of seconds in which modifications should be detected n/a Yes

 

<cron/>

The Cron Task lets you specifiy a timer for monitoring repository by the <modificationset/>.

<project name="ProjectName">
    <trigger>
        <cron timer="*/5 * * * *"/>
    </trigger>
...
</project>

Parent Tags

  • <trigger/>

Parameters

Name Type Description Default Required
timer String Cron like configuration of timer n/a Yes

<sensor/>

The Sensor Task lets you specifiy a file for monitoring to start a build. If file exists the build will start and the file will be removed.

<project name="ProjectName">
    <trigger>
        <sensor file="path/to/file"/>
    </trigger>
...
</project>

Parent Tags

  • <trigger/>

Parameters

Name Type Description Default Required
file String Path to a file. n/a Yes
