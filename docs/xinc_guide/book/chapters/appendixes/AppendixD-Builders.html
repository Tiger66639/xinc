Appendix D: Builders

This appendix contains a reference of all builder tasks, i.e. all tasks that can run build scripts. At the moment Xinc uses only a Phing implementation.

<phingbuilder/>

Parent Tags

  • <builders/>
<project name="ProjectName">
...
    <builders>
        <phingbuilder buildfile="build.xml" target="main"/>
    </builders>
...
</project>

The above example will execute the Phing buildscript "build.xml" and call the target "main".

Parameters

Name Type Description Default Required
buildfile String Filename of the Phing buildfile n/a Yes
target String Target inside the Phing buildscript which shall be called n/a No
params String Extra parameters which will be passed to phing when executing it from the command line n/a No
workingdir String Setting the working directory before executing Phing n/a No
if String Name of property to check for boolean == true. If this property name is true, the phingbuilder will be processed. n/a No

All Properties that are available in Xinc will be passed to the Phing buildscript with the prefix "xinc.".

Example:
build.number => xinc.build.number
